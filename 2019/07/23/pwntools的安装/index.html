ubuntu16下 pwntools的安装 

sudo apt-get update
sudo apt-get install python2.7 python-pip python-dev git libssl-dev libffi-dev build-essentia
sudo pip install --upgrade pip -i http://pypi.douban.com/simple --trusted-host pypi.douban.com
sudo pip install --upgrade pwntools  -i http://pypi.douban.com/simple --trusted-host pypi.douban.com

如果第四条指令报错的话，可以在最后加上 –user 试试；
